## Git, historia y motivación

Git se ha convertido en el estándar de production academia e industrial en los últimos anos. Por el trasfondo de la tecnología, aunque aparezcan nuevas alternativas para utilizar git (clientes), el trasfondo tecnológico prevalecerá durante varios anos/décadas convirtiéndose asi en una herramienta practica y estratégica que conviene saber utilizar.
